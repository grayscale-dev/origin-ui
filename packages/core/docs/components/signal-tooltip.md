# signal-tooltip

Tooltip / hovercard component.

## Props

- `content: string`
- `variant?: "tooltip" | "hovercard"` (default `"tooltip"`)
- `placement?: "top" | "right" | "bottom" | "left"` (default `"top"`)
- `delay?: number` (default `150`)
- `interactive?: boolean`
- `disabled?: boolean`

## Slots

- default — trigger
- `content` — custom body
