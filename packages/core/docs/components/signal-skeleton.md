# signal-skeleton

Skeleton/placeholder and empty/error panel.

## Props

- `variant?: "skeleton" | "empty" | "error"` (default `"skeleton"`)
- `lines?: number` (default `3`)
- `media?: boolean`
- `heading?: string` (attribute: `title`)
- `description?: string`

## Slots

- `title`, `description`, `actions` (for empty/error)
